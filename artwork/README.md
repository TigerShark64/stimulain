# Stimulain Master Artwork Folder

Place your track cover artwork images inside this folder.

## Suggested File Names:
1. `rohaan-easy-for-them.jpg` (Artwork for the Rohaan Remix)
2. `late-2025-mix.jpg` (Artwork for the Late 2025 Mix)

## Technical Details:
* **Format**: JPG or PNG
* **Dimensions**: Square aspect ratio (minimum 500x500 pixels recommended for high-res rendering)
* **Fallback**: If this folder is empty or images are missing, the website will automatically load abstract fallback graphics from our Unsplash visual servers.
