# Stimulain Master Audio Tracks Folder

Place your native master MP3 or WAV files inside this folder.

## How to name and link your files:
1. Copy your audio files (e.g. `dark-frequency.mp3`) into this directory.
2. Open [js/projects-data.js](file:///c:/Users/equin/OneDrive/Documents/Antigravity%20Dark%20Matter%20Projects/js/projects-data.js) (or directly in your script files) if you want to update the tracks playlist.
3. Refresh the page to see them registered in your custom player!

*Note: A synthesized MIDI oscillator fallback is implemented in JavaScript. If this folder is empty, clicking Play on the site will automatically generate a pulsing techno note so that the spectrum analyzer is fully functional out-of-the-box!*
